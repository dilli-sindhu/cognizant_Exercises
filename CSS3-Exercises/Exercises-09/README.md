# Exercise 9: Multiple Columns in Text

## Scenario

The community bulletin needs to be displayed like a newspaper.

## Objective

Use CSS3 multi-column layout.

## Tasks Completed

### 1. Created a News Article Section

```html
<section class="news-article">
    <p>Article Content...</p>
</section>
```

### 2. Applied Multiple Columns

```css
.news-article {
    column-count: 2;
}
```

### 3. Added Gap Between Columns

```css
column-gap: 30px;
```

### 4. Added Divider Line

```css
column-rule: 1px solid gray;
```

## Concepts Practiced

- CSS Multi-Column Layout
- column-count
- column-gap
- column-rule
