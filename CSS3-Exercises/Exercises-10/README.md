# Exercise 10: Responsive Web Design with Media Queries

## Scenario

Users will access the portal on phones, tablets, and desktops.

## Objective

Apply media queries for responsiveness.

## Tasks Completed

### 1. Added Media Query

```css
@media screen and (max-width: 768px) {
}
```

### 2. Stacked Navigation Vertically

```css
.nav-menu {
    flex-direction: column;
}
```

### 3. Reduced Font and Image Sizes

```css
h1 {
    font-size: 24px;
}

.eventCard img {
    width: 80%;
}
```

### 4. Used Flexible Units

```css
width: 48%;
padding: 2%;
font-size: 3vw;
height: 100vh;
```

### 5. Used Flexbox Layout

```css
.events-container {
    display: flex;
}
```

## Concepts Practiced

- Media Queries
- Responsive Design
- Flexbox
- Percentage Units (%)
- Viewport Width (vw)
- Viewport Height (vh)
