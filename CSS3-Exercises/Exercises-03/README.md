# Exercise 3: Selectors Playground

## Scenario
You need to style various elements based on IDs, classes, and element types.

## Objective
Master different selector types.

## Tasks Completed

### 1. Universal Selector

Used `*` to reset margin and padding.

```css
* {
    margin: 0;
    padding: 0;
}
```

### 2. Element Selector

Styled all `<h2>` elements.

```css
h2 {
    color: darkblue;
}
```

### 3. ID Selector

Styled the banner using `#mainHeader`.

```css
#mainHeader {
    background-color: #007BFF;
}
```

### 4. Class Selector

Styled event containers using `.eventCard`.

```css
.eventCard {
    padding: 15px;
}
```

### 5. Grouping Selector

Applied the same style to both `<h3>` and `<p>`.

```css
h3,
p {
    color: #444;
}
```

## Concepts Practiced

- Universal Selector (`*`)
- Element Selector
- ID Selector (`#`)
- Class Selector (`.`)
- Grouping Selector (`,`)
