# Exercise 5: Typography – Fonts and Text

## Scenario
The marketing team wants more appealing fonts and better readability.

## Objective
Enhance textual appearance using CSS properties.

## Tasks Completed

### 1. Included a Google Font

```html
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600&display=swap" rel="stylesheet">
```

### 2. Applied Typography Properties

```css
font-family
font-size
font-style
font-weight
```

### 3. Styled Text

```css
text-align
text-transform
letter-spacing
line-height
```

## Concepts Practiced

- Google Fonts
- Font Family
- Font Size

- Font Weight
- Text Alignment
- Text Transformation
- Letter Spacing
- Line Height
