# Exercise 4: Color & Background Styling

## Scenario
You’re theming the portal based on a city council’s branding.

## Objective
Apply consistent colors and background visuals.

## Tasks Completed

### 1. Used HEX Colors

```css
h2 {
    color: #003366;
}
```

### 2. Used RGBA Colors

```css
.event-section {
    background-color: rgba(255, 255, 255, 0.8);
}
```

### 3. Added Background Image with Fallback Color

```css
body {
    background-color: #e6f2ff;
    background-image: url("city-background.jpg");
}
```

### 4. Applied Linear Gradient

```css
.section-header {
    background: linear-gradient(
        to right,
        #007BFF,
        #00C6FF
    );
}
```

## Concepts Practiced

- HEX Colors
- RGBA Colors
- Background Images
- Fallback Background Color
- Linear Gradients
